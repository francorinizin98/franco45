Herramientas/Mosaiquismo


Tenaza

Se utiliza para cortar las venecitas.

Pinza con punta curva

Se utiliza para el corte rapido de espejos,vidrios,etc.

Puntas de corte

Se utiliza para marcar los espejos,vidrios,etc. Despues de hacer las marca se utiliza la pinza de punta curva para cortarlos facilmente.


Herramientas/Pintura


Pinceles

Se utiliza para pintar con acrilico.

Espatulas

Se utiliza para pintar con oleo.

Atril

Fundamental para poder trabajar comodos al momento de expresarse con el arte de la pintura.


Herramientas/Decoupage


Servilletas

Unas de las herramientas mas facil de conseguir y a la vez muy economico.
